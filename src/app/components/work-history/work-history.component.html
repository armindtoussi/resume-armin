Open Data Developer Network

2017-present

Web App Developer

Open Data Developer Network, or ODEN for short, is an open data concept who's mission is to simplify the way developers get open data. Governments and companies are releasing open data in large amounts these days, however, there is no standardization and that makes it harder for developers to use the datasets. It's time consuming for developers to write converters for each individual dataset. ODEN is trying to change that. As my second job in the tech sector, I was responsible for designing the structure of the web app as well as coding a large amount of the web application using Angular 4.

  • Responsible for designing application structure
  • Communicated effectively using Slack channels, voice calls, and in person meetings
  • Responsible for managing Git repository on Bit Bucket and managing release standards
  • Adhered to clean coding standards
  • Connected and worked with Firebase and Firestore databases
  • Built modular application using Angular 4 component structure
  • Adhered to website UI design
